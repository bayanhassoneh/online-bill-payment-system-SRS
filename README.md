# online-bill-payment-system-SRS-document
A Software Requirements Specification (SRS) for an Online Bills Payment System

This project provides a comprehensive **Software Requirements Specification (SRS)** and **System Design**.It allows users to pay bills via bank accounts or electronic wallets.

Development
